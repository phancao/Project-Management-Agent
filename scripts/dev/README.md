# Development Helper Scripts

This directory contains helper scripts for development and debugging.

## Scripts

- `start_backend_with_logs.sh` - Start backend server with logging
- `tail_backend_logs.sh` - Tail backend log files

## Usage

Run scripts from the project root:

```bash
# From project root
bash scripts/dev/start_backend_with_logs.sh
bash scripts/dev/tail_backend_logs.sh
```

Or from within this directory:

```bash
cd scripts/dev
bash start_backend_with_logs.sh
```

## Notes

- These scripts are development tools
- They may require specific environment setup
- Check individual script comments for usage details
